# &nbsp;Sziget-Hódító Stratégia

# Egy körökre osztott, taktikai asztali stratégiai játék, amely HTML5, CSS3 és JavaScript (jQuery) technológiák felhasználásával készült a Multimédia kurzus keretein belül.

# 

# &nbsp;Alapötlet

# A játék egy rácsalapú térképen játszódik, ahol két játékos küzd egymás ellen ugyanazon a számítógépen (időosztásos multiplayer). A cél erőforrások gyűjtése, egységek gyártása és az ellenfél legyőzése taktikázással.

# 

# &nbsp;Játékmenet alapjai

# Körökre osztott rendszer: A játékosok felváltva cselekedhetnek. Egy kör akkor ér véget, ha a játékos megnyomja a "Kör vége" gombot.

# 

# Térkép: A játéktér fix méretű rácsokból áll, amelyeken különböző tereptárgyak (pl. aranybányák) találhatók.

# 

# Erőforrás-kezelés: A játékosok aranyat gyűjthetnek a térképen található bányákból, amiből új egységeket hozhatnak létre.

# 

# &nbsp;Egységtípusok (Tervezet)

# Munkás: Erőforrások begyűjtésére és építkezésre szolgál.

# 

# Katona: Lassabb mozgású, de erős támadóegység az ellenfél legyőzésére.

# 

# Felderítő: Gyors mozgású egység a térkép felfedezéséhez.

# 

# &nbsp;Technikai megvalósítás

# Keretrendszer: jQuery a DOM manipulációhoz és eseménykezeléshez.

# 

# Grafika: CSS alapú rácsháló és abszolút pozicionált egységek.

# 

# Animáció: Az egységek mozgása jQuery .animate() funkcióval történik.

# 

# Adattárolás: A játékállás mentése a böngésző localStorage területére készül.
